# Privacy Policy

Last updated: 2026-02-24

Rabbit vs Mecha Calculation Game（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーでは、本アプリにおける情報の取り扱いについて説明します。

## Information Collection

本アプリは、ユーザーの個人情報（氏名、住所、メールアドレスなど）を収集しません。

## Automatically Collected Information

本アプリは、ユーザーを特定できる情報を収集しません。  
ただし、アプリの改善のために、デバイス情報やクラッシュ情報などがOSによって自動的に収集される場合があります。

## Third-Party Services

本アプリは第三者サービスを利用しないか、ユーザー情報を収集する目的で第三者に提供することはありません。

## Children’s Privacy

本アプリは13歳未満の子どもを対象として個人情報を収集することはありません。

## Data Security

本アプリは個人情報を収集しないため、ユーザーの個人データを保存・管理することはありません。

## Changes to This Privacy Policy

本ポリシーは必要に応じて更新される場合があります。変更があった場合、本ページにて告知します。
